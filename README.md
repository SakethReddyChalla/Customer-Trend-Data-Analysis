# 📊 Customer Behavior Analysis Dashboard

## 📌Overview
This project focuses on analyzing **customer shopping behavior** using data analytics techniques.  
It helps understand customer patterns, purchasing trends, and preferences to support better business decisions.

The analysis was performed using **SQL for querying, Python for data preprocessing, and Power BI for visualization**.  
The dashboard provides insights into customer segments, spending behavior, and product trends.

---

## 📊 Key Insights
- Customer purchase patterns
- Spending behavior analysis
- Product/category trends
- Customer segmentation
- Region-wise insights
